## 🎮 游戏功能

### 核心玩法

1. **选择病原体** - 5 种类型可选
2. **进化传播** - 5 类传播途径
3. **发展症状** - 6 级症状进化
4. **强化能力** - 5 种特殊能力
5. **感染全球** - 30 个真实国家

### 游戏命令

```
start          - 开始新游戏
load           - 读取存档
help           - 显示帮助
status         - 查看状态
upgrade        - 查看升级选项
upgrade [名称]  - 执行升级
save           - 保存游戏
pause          - 暂停/继续
```

## 📁 项目结构

```
sense/
├── src/                      # 源代码
│   ├── core/                 # 核心引擎
│   │   ├── GameState.js      # 游戏状态
│   │   ├── GameEngine.js     # 游戏引擎
│   │   └── SaveManager.js    # 存档管理
│   ├── systems/              # 游戏系统
│   │   ├── PathogenSystem.js # 病原体系统
│   │   └── CountrySystem.js  # 国家系统
│   ├── ui/                   # 用户界面
│   │   ├── TextRenderer.js   # 文字渲染
│   │   └── InputHandler.js   # 输入处理
│   └── main.js               # 主入口
├── assets/                   # 资源文件
├── config/                   # 配置文件
├── tests/                    # 测试文件
├── doc/                      # 文档
├── css/                      # 样式
├── public/                   # 公共资源
├── index.html                # 主页面
├── package.json              # 项目配置
├── vite.config.js            # Vite 配置
├── .eslintrc.json            # ESLint 配置
├── .env.development          # 开发环境
├── .env.production           # 生产环境
├── README.md                 # 项目说明
└── REFACTORING_PLAN.md       # 重构计划
```

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问游戏

打开浏览器访问：http://localhost:3000

### 4. 开始游戏

```
输入 "start" 开始
输入 "1-5" 选择病原体
输入 "upgrade" 查看升级
```
