#### 核心引擎 (src/core/)

| 文件名 | 用途 | 负责人 | 状态 |
|--------|------|--------|------|
| GameState.js | 游戏状态管理，包含所有游戏数据的序列化/反序列化 | 核心开发 | ✅ 完成 |

## 目录结构说明

```
sense/
├── src/                      # 源代码目录 - 所有游戏逻辑代码
│   ├── core/                 # 核心引擎模块
│   ├── systems/              # 游戏系统模块
│   ├── ui/                   # 用户界面模块
│   └── main.js               # 程序入口
├── assets/                   # 资源文件目录 - 图片、音频、字体等
├── config/                   # 配置文件目录 - 环境配置
├── tests/                    # 测试文件目录 - 单元/集成/性能测试
├── doc/                      # 文档目录 - API、设计、指南
├── css/                      # 样式文件目录
├── public/                   # 公共资源目录
├── index.html                # 主页面
├── package.json              # 项目配置
├── vite.config.js            # Vite 配置
├── .eslintrc.json            # ESLint 配置
├── .gitignore                # Git 忽略配置
├── .env.development          # 开发环境变量
├── .env.production           # 生产环境变量
├── README.md                 # 项目说明文档
└── REFACTORING_PLAN.md       # 重构计划文档
```
