# Pathogen Inc. - 病原体公司

一款类似《瘟疫公司》的策略类网页游戏。扮演致命病原体，通过进化传播途径、症状和能力来感染全人类！

## 🎮 游戏玩法

1. **选择病原体** - 细菌、病毒或寄生虫，每种都有独特的属性
2. **进化传播途径** - 空气、水源、血液、接触、动物传播
3. **进化症状** - 从轻微咳嗽到致命器官衰竭
4. **强化能力** - 抗寒性、抗热性、抗药性、快速变异
5. **征服世界** - 感染全人类，战胜各国防控措施

## 🛠️ 技术栈

- HTML5
- CSS3 (现代布局 + 动画效果)
- JavaScript (原生，无框架依赖)
- SVG (世界地图可视化)

## 🚀 运行方式

直接用浏览器打开 `index.html` 即可游玩：

```bash
# 使用简单 HTTP 服务器
python -m http.server 8000
# 或
npx serve .
```

然后访问 http://localhost:8000

## 📦 部署

已配置 GitHub Pages，访问 https://your-username.github.io/pathogen-inc 即可在线游玩。

## 📋 功能特性

- 25个国家/地区，每个都有独特的防控能力
- 随机事件系统（医疗突破、自然灾害、防控变化等）
- 实时感染进度可视化
- 动态难度调整
- 响应式设计，支持移动端
